Para acceder a las variables de entorno en el código nativo de Android, sigue estos pasos:

En tu archivo android/app/src/main/java/com/{nombre_del_proyecto}/MainApplication.java, importa la clase Config:

java
Copy code
import com.lugg.ReactNativeConfig.ReactNativeConfigPackage;
Luego, agrega la instancia de ReactNativeConfigPackage a la lista de paquetes devueltos en el método getPackages():

java
Copy code
protected List<ReactPackage> getPackages() {
  return Arrays.asList(
      new MainReactPackage(),
      new ReactNativeConfigPackage()
  );
}
Ahora puedes usar las variables de entorno en tu código Java:

java
Copy code
import com.lugg.ReactNativeConfig.ReactNativeConfig;

String apiUrl = ReactNativeConfig.getString("API_URL");
String apiKey = ReactNativeConfig.getString("API_KEY");
